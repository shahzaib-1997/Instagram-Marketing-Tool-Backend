Story Actions
  • What algorithm type do you want to use for story likes?
  • 1:1 - only works together with view stories action, using this algorithm type you will like and view story only together, separate like or view will be not allowed
  • 1:1 + M - it's the same as algorithm 1:1, but you will be able to continue viewing stories after story like limit reached
  • 1:1 + M (last story only) - it's the same as algorithm 1:1 + M, but you will be able to mark as seen only last story of user
  • Basic - like story and skip, while story like delay is not expired
  • Advanced - like story and pause automation, while story like delay is not expired
  • Advanced (view last story only) - it's the same as algorithm Advanced, but you will be able to mark as seen only last story of user
